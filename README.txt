Data Source - https://grouplens.org/datasets/movielens/

Two recommendation system
1.Boltzmann machine - That recommendation system predict if user going like Yes or no a movie.(Predict binary outcome)
  we obtained an Average Distance of 0.24, which is equivalent to about 75% of correct prediction.
2.AutoEncoder - That recommendation system predict the rating of a movie by user.(Predict rating from 1 to 5)
  The test loss 0.95 star, that is less than one star
